Приложение для совместного списка покупок в реальном времени.

## Технологии
- Frontend: React, Vite, Tailwind CSS, Socket.IO Client, Zustand
- Backend: Node.js, Express, TypeScript, Socket.IO, MongoDB, OpenAI API

## Установка
```bash
cd backend
npm install
npm run dev
cd ../frontend
npm install
npm run dev
